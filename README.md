🐾 Sistema de Gestión para Hospital Veterinario
📌 Descripción

Este proyecto implementa un sistema básico orientado a objetos en Python para la gestión de un hospital veterinario. Permite administrar personas, mascotas, consultas, tratamientos, facturación y métodos de pago, aplicando principios fundamentales de la Programación Orientada a Objetos (POO).

🎯 Objetivo

Diseñar e implementar un sistema funcional que modele el funcionamiento de una clínica veterinaria, aplicando:

Herencia
Polimorfismo
Encapsulación
Abstracción
Relaciones entre clases (asociación, composición, agregación)

⚙️ Funcionalidades

✔ Registro de clientes y mascotas
✔ Creación de consultas veterinarias
✔ Asignación de tratamientos
✔ Cálculo automático de facturas
✔ Soporte para múltiples métodos de pago
✔ Simulación del proceso completo del sistema
